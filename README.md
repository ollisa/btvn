# btvn

### Overview
This is a repository for 'Bayesian Estimation of Sparse Partial Correlation Graphs with Smooth Temporal Evolution'

### Purpose
We propose an approach to estimate a smoothly evolving chain of networks using sparse partial-correlation graphs in a Bayesian framework.
Models are implemented using [CmdStanR](https://mc-stan.org/cmdstanr/articles/cmdstanr.html).
Scripts for various network estimation methods are provided, along with a working example and tutorials for our BTVN approach.

### FYI
Instead of the TVGLASSO implementation provided here, we recommend checking the R package [rTVGL](https://github.com/THautamaki/rTVGL).
